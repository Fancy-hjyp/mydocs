# 批量新增
使用Mybatis的标签进行批量新增操作
```xml
<insert id="insertUserBatch" parameterType="java.util.List">
    insert into tb_user
    <trim prefix="(" suffix=")" suffixOverrides=",">
        <if test="list[0].name != null">
            name,
        </if>
        <if test="list[0].age != null">
            age,
        </if>
    </trim>
    values
    <foreach collection="list" item="item" separator=",">
        <trim prefix="(" suffix=")" suffixOverrides=",">
            <if test="item.name != null">
                    #{item.name},
            </if>
            <if test="item.age != null">
                    #{item.age},
            </if>
        </trim>
    </foreach>
</insert>
```
借助于trim和if标签，可以动态确定插入哪些字段；同时借助于foreach标签，可以批量插入数据。
